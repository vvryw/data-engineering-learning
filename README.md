# Data Engineering Learning Notes

This repository contains my learning notes, exercises, and hands-on projects from the course **Data Engineering for Beginners: Learn SQL, Python & Spark**.

The goal of this repository is to document my learning journey and build a practical foundation in Data Engineering using **SQL**, **Python**, **PostgreSQL**, **PySpark**, **Spark SQL**, **Databricks**, and **GCP Dataproc**.


## Topics

| Topic | Description |
|---|---|
| SQL | Querying, filtering, joins, aggregations, indexes, functions, and database essentials |
| Python | Programming fundamentals, collections, Pandas, file processing, and database programming |
| PySpark | Spark DataFrame APIs, Spark SQL, transformations, actions, and ETL pipelines |
| PostgreSQL | Tables, indexes, SQL queries, built-in functions, and database operations |
| Databricks | Practice environment for PySpark and Spark SQL |
| GCP Dataproc | Hadoop and Spark cluster setup for distributed data processing |
