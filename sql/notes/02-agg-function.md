# Aggregate Functions

## Summary

Aggregate functions for SQL queries

## SUM
Use SUM() to sum of something (value should be integer or float)
- Use round( SUM(...) , 2) to round 2 decimal place

## GROUP BY
Use GROUP BY to group things in common

## HAVING 
Use having to filter something from GROUP BY (cannot use WHERE)

## to_char()
To change format of data
- to_char(order_date, 'yyyy-MM') AS order_month
